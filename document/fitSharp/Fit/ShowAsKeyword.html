Show As Keyword
The Show As keyword is one of the Flow Fixture Keywords. It executes a method on the System Under Test and displays the result using one or more cell attributes.

The Raw attribute displays the result directly with no encoding.

when test is
sample do
settext<b>stuff</b>
show asrawtext
then result is
sample do
settext<b>stuff</b>
show asrawtext stuff

The Formatted attribute displays the result encoded with formatting preserved.

when test is
sample do
settext<b>more</b>
stuff
show asformattedtext
then result is
sample do
settext<b>more</b>
stuff
show asformattedtext 
<b>more</b>
stuff

The Folded attribute displays the result in a collapsible section.

when test is
sample do
settextstuff
show asfoldedtext
then result is
sample do
settextstuff
show asfoldedtext ⇓
stuff
